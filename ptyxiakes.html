Διαδικασία, θεματικές περιοχές και γενικές παρατηρήσεις

  • Τι θα φτιάξετε: Οι εργασίες έχουν ως στόχο την παραγωγή πλήρως λειτουργικών εφαρμογών. Για την επιτυχή ολοκλήρωση της πτυχιακής θα πρέπει να παρουσιάσετε μια άρτια και ελεγμένη εφαρμογή με πλήρη τεκμηρίωση. Οι θεματικές περιοχές στις οποίες κινούνται οι εργασίες είναι (ενδεικτικά):
    • Εφαρμογές που βασίζονται σε σημασιολογικά δεδομένα-linked data ή δεδομένα GIS.
    • Εφαρμογές που εκμεταλλεύονται αλγορίθμους μηχανικής μάθησης και AI.
    • Εφαρμογές που συνδυάζουν πολλαπλά web APIs και backends.
    • Εφαρμογές web που χρησιμοποιούν μοντέρνες μεθοδολογίες και πρακτικές (single page applications, χρήση client-side και UI frameworks, webAssembly κλπ.)
  • Πώς θα το φτιάξετε; Αρχικά πρέπει να έχετε μια δική σας πρωτότυπη ιδέα: δεν χρειάζονται τεχνικές λεπτομέρειες, αρκεί το τι θέλετε να κάνετε (π.χ. «θέλω να μελετήσω την επίδραση των ακτίνων γάμμα στις μαργαρίτες»). Αποφύγετε τετριμμένες ιδέες (π.χ. «ένα e-shop»). Παρουσιάστε την ιδέα σας και, εφόσον συμφωνηθεί η εκπόνηση της πτυχιακής εργασίας πάνω σε αυτήν, θα σας δοθούν οι απαραίτητες συμβουλές και τεχνικές οδηγίες για να προχωρήσετε.
  • Τι χρειάζεται; Η καλή γνώση προγραμματισμού είναι αναγκαία για το σύνολο των πτυχιακών εργασιών. Αυτό, ανεξάρτητα από τη γλώσσα προγραμματισμού· το πιο πιθανό είναι να χρησιμοποιήσετε γλώσσες προγραμματισμού και βιβλιοθήκες πέραν εκείνων που διδάσκονται στα πλαίσια των μαθημάτων! Θα δοθεί έμφαση στις web-based εφαρμογές. Απαραίτητη είναι επίσης η χρήση λογισμικού και μεθοδολογιών ανοικτού κώδικα.
